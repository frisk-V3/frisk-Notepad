# 📘 friskOS Notepad  
friskOS に標準搭載される予定の、軽くて速くて気持ちいいテキストエディタ。

Windows メモ帳のシンプルさをベースにしつつ、  
frisk の性格・世界観・色を取り入れた “frisk流メモ帳” です。

---

## ✨ 特徴

- 🖊 **ソースコード系ファイルも読み込み可能**  
  `.txt .md .cs .py .html .c .cpp .js .json .fs` など幅広く対応

- 💾 **Ctrl+S で保存**  
  ブラウザでもアプリ化しても動くショートカット

- ↩️ **元に戻す（Undo）対応**  
  メニューにもあるし、もちろん **Ctrl+Z が最強**

- 🧵 **折り返し ON/OFF**  
  コードも文章も読みやすく

- 🖥 **アプリ化を意識したUIサイズ**  
  メニューバーは “大きすぎず小さすぎず” の絶妙バランス

---

## 🎨 アプリアイコン（frisk Notepad）

frisk の色を取り入れた、  
透明背景の OS 標準アプリ風アイコンを採用。

## 📂 対応ファイル形式

.txt .md .cs .py .html .htm .c .cpp .h .js .ts .json .fs .css .xml .yml .yaml

## 🛠 開発の背景

標準アプリとして軽くて気持ちよく動くメモ帳が必要だった。

そのために作られたのが **friskOS Notepad**。

- シンプル  
- 速い  
- 使いやすい  
- frisk の世界観に合う  

これらを満たすように設計されています。
